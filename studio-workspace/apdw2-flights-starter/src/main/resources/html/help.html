Flights Search

Submit a request to http://localhost:8081/flights?code=<>&airline=<>&simulate=<>

Default Values are:

  • code: SFO
  • airline: american
  • simulate: true

If simulate = true, then all three airline services are simulated locally within the Mule application. This allows the Flights search to work when the computer is offline.

Here are some common links used in the course:

  • http://locahost:8081/flights?code=SFO&airline=american
  • http://locahost:8081/flights?code=LAX&airline=delta
  • http://locahost:8081/flights?code=CLE&airline=united
  • http://locahost:8081/flights?code=LAX&airline=all
